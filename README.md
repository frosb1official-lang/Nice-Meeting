# Nice Meeting 💕

> Милое приложение, которое **не принимает ответ «Нет»** ❤️

![Nice Meeting Preview](https://via.placeholder.com/800x450/f26389/ffffff?text=Nice+Meeting+Preview)

## ✨ О проекте

Интерактивное приложение-свидание с милыми анимациями, убегающей кнопкой «Нет» и тёплой атмосферой.  
Сделано с душой и любовью.

### Особенности
- ✅ Кнопка «Нет» убегает от курсора
- ✅ Красивые анимированные гифки
- ✅ Собрано через **Tauri** — маленькое и быстрое приложение
- ✅ Работает на Windows

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/твой-username/nicemeeting.git

# Перейти в папку проекта
cd nicemeeting

# Установить зависимости
npm install

# Запустить приложение
npm run tauri dev

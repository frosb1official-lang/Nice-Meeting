# Nice Meeting 💕

> Милое приложение, которое **не принимает ответ «Нет»** ❤️

## ✨ О проекте

Интерактивное приложение-свидание с милыми анимациями, убегающей кнопкой «Нет» и тёплой атмосферой.  
Сделано с душой и любовью.

### Особенности
- ✅ Кнопка «Нет» убегает от курсора
- ✅ Красивые анимированные гифки
- ✅ Адаптивный веб-интерфейс
- ✅ Работает на любых устройствах через браузер

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/твой-username/nicemeeting.git

# Перейти в папку проекта
cd nicemeeting
```

Для запуска приложения локально:
- Откройте файл `src/index.html` напрямую в браузере.
- Или используйте расширение **Live Server** в VS Code.

## 🌐 Развертывание

Так как приложение состоит из статических HTML, CSS и JavaScript файлов, его можно легко разместить на любом хостинге статических сайтов (GitHub Pages, Netlify, Vercel) или на собственном веб-сервере (Nginx, Apache). Просто скопируйте содержимое папки `src` в корневую директорию вашего веб-сервера.
